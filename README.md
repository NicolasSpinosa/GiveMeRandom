# Description

**GiveMeRandom** is an android app build using **React Native CLI** meant to give its users reliable tools of random nature, such as picking a number, elements from lists, etc.

# Motivation

While playing some mobile games, I like to add spice by making decisions based on random chance, but all the offline apps I've tried fall in one of the following categories:
- The generation method *seemed* to generate poorly distributed randomness.
- They aren't comfortable to use.
- The UI is not pleasing to look at.

Although only the first point should be of upmost importance, this was enough for me to decide to build my own app, ensuring that it doesn't fall in any of those categories.

# Technologies

- **Main framework:** React Native CLI.
- **State management:** Zustand.
- **UI:** Unwind.
- **Non-Encrypted storage:** MMKV.
